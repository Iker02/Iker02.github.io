Explorando en CSS

Introducción

La propiedad object-position en CSS se utiliza para controlar la posición de un elemento de imagen cuando se utiliza la propiedad object-fit. Esta propiedad es útil para controlar cómo se muestra una imagen dentro de un contenedor con dimensiones específicas.
